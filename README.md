## 📌 Overview
The **AI Virtual Mouse Using Hand Gestures** is a machine learning-based human-computer interaction system that allows users to control the mouse using hand gestures captured through a webcam.

It uses **computer vision (OpenCV + MediaPipe)** and **deep learning-based hand tracking models** to detect hand landmarks and convert them into real-time mouse operations such as movement, clicking, and gesture-based control.

This project demonstrates how AI can replace traditional input devices and make computing more natural and accessible.

---

## ⚙️ Features
- ✋ Real-time hand tracking using webcam  
- 🖱️ Control mouse cursor with finger movement  
- 👆 Left-click and right-click gestures  
- 🎯 Smooth and accurate cursor movement  
- 💡 No physical mouse required  
- 🚀 Lightweight and beginner-friendly AI project  

---

## 🧠 Tech Stack / Dependencies

Install the required libraries:

- 📷 **OpenCV** → Image processing & computer vision  
- ✋ **MediaPipe** → Hand tracking & landmark detection  
- 🖱️ **AutoPy** → Mouse control automation  
- 🔢 **NumPy** → Numerical computations
